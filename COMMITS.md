# Commits

## Categories

Every commit's message should be preceded by an emoji representing the category of task this commit belongs to.
The list of such emojis and associated categories is the following:

- 📚: Thesis
- 🤖: Agda code
- 🏗️: Build system
- 🚚: Continuous delivery
- ℹ️: Documentation
- 💄: Aesthetics
- ♻️: Refactoring
- 📦: Meta
